Introduction

Thank you for purchasing Fine Dining Restaurant Platform. This is a complete solution for modern restaurants, featuring a public reservation website, an immersive "Table Companion App" for guests, and a comprehensive Admin Dashboard for staff.

Designed with Next.js 16 and Tailwind CSS v4, it offers blazing fast performance and SEO optimization out of the box.

Key Features

  • Dynamic Reservation System: Real-time booking with deposit calculation.
  • Table Companion App: Guests can view menus and request service via QR code.
  • Kitchen Display System (KDS): Real-time order tracking for kitchen staff.
  • Email Notifications: Automated confirmations via Resend.
  • Storytelling Menu: Organize dishes by "Chapters" for a fine dining experience.

Prerequisites

Before you begin, ensure you have the following installed:

  • Node.js (v20.0.0 or higher)
  • PostgreSQL (Local installation or Cloud provider like Neon/Supabase)
  • Git

Local Installation

Follow these steps to run the project on your local machine.

1. Unzip and Install

cd your-project-folder
npm install

2. Configure Environment

Rename .env.example to .env and fill in your details.

cp .env.example .env

3. Automatic Setup

We have included a setup script to initialize the database and seed dummy data.

npm run setup

This command will:
- Generate Prisma Client
- Push Schema to Database
- Seed initial data (Menu, Tables, Admin User)

4. Start Development Server

npm run dev

Open http://localhost:3000 to view the app.

Changing Branding

You can easily update the restaurant name, slogan, and contact info without touching the code.

Open file: src/config/site.ts

export const siteConfig = {
  name: "Heritage", // Change to your Restaurant Name
  description: "Experience the finest culinary journey...",
  contact: {
    phone: "+62 812 3456 7890",
    email: "reservations@your-restaurant.com"
  }
  // ...
}
